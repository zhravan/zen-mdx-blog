## What It Does

The external-links plugin automatically:

- **Adds icon** - Shows ↗ symbol for external links
- **Opens in new tab** - Uses `target="_blank"`
- **Security attributes** - Adds `rel="noopener noreferrer"`
- **Optional nofollow** - Can add `rel="nofollow"` if configured

## Configuration

The plugin is configured in `config/plugins.yaml`:

```yaml
external-links:
  enabled: true
  icon: "↗"           # Icon to show
  openInNewTab: true  # Open externals in new tab
  nofollow: false     # Add nofollow attribute
```

## Security

External links automatically get these attributes for security:

- `target="_blank"` - Opens in new window
- `rel="noopener"` - Prevents window.opener access
- `rel="noreferrer"` - Doesn't send referrer information

This protects against:
- Reverse tabnabbing attacks
- Window.opener exploits
- Privacy leaks
